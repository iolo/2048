# 2048

Clones of [2048 Game](https://en.wikipedia.org/wiki/2048_\(video_game\)) for Old Apple IIx Series.

> NOTE: Extremely sloooooow. Use Emulator runs on 4Mhz+.

- [text2048](text2048.abas): TEXT(40x24 mono) mode
- [gr2048](gr2048.abas): Lo-res Graphics(40x48 16color) mode
- [grx2048](grx2048.abas): Lo-res Graphics mode with my own [Assemlby Module](grx.s)
- [dgr2048](dgr2048.abas): Double Lo-res Graphics(80x48 16color) mode with [DOLORES Library](https://github.com/mgolombeck/DOLORES)
- [dgrx2048](dgrx2048.abas): Double Lo-res Graphics mode with my own [Assembly Module](dgrx.s)
