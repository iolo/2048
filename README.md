# 2048

Clones of [2048 Game](https://en.wikipedia.org/wiki/2048_\(video_game\)) for Old Apple IIx Series.

> NOTE: Extremely sloooooow. Use Emulator runs on 4Mhz+.

- [text2048](text2048.abas): TEXT(40x24 mono) mode
- [gr2048](gr2048.abas): Lo-res Graphics(40x48 16color) mode
- [grx2048](grx2048.abas): Lo-res Graphics(40x48 16color) mode with [Assemlby Module](grx.s)
- [dgr2048](dgr2048.abas): Double Lo-res Graphics(80x48 16color) mode.
    - using [DOLORES from 8bitshack](https://www.8bitshack.net/dolores-info) that was based on [DOLORES by Golombeck](https://github.com/mgolombeck/DOLORES)
- TODO: hgr2048: HI-RES Graphics(280x192 mono or 140x192 6color) mode.
- TODO: dhgr2048: Double Hi-res Graphics(560x192 mono or 140x192 16color) mode
